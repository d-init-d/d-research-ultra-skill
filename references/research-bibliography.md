# Research Bibliography and Design Basis

This skill is based on common mechanisms from browser automation, web crawling, information retrieval, and evidence synthesis.

## Browser automation and scraping

- Playwright documentation: https://playwright.dev/docs/intro
- Playwright GitHub: https://github.com/microsoft/playwright
- Scrapy documentation: https://docs.scrapy.org/en/latest/
- Crawlee documentation: https://crawlee.dev/docs/introduction

## Web discovery standards

- Robots Exclusion Protocol RFC 9309: https://datatracker.ietf.org/doc/html/rfc9309
- Sitemaps protocol: https://www.sitemaps.org/protocol.html

## Information retrieval mechanisms

- Query expansion
- Boolean retrieval
- pseudo-relevance feedback
- snowballing and citation pearl growing
- source triangulation

## Academic research mechanisms

- PRISMA 2020 statement and checklist: https://www.prisma-statement.org/
- PRISMA-S extension for search reporting
- IMRaD structure for academic writing
- snowballing literature review methods

## Agent safety and governance themes

- skill/package supply-chain risks
- prompt injection in retrieved content
- data exfiltration risks
- read-only defaults
- blocker reports instead of access-control bypass
